## Структура SumTree для Prioritized Replay

```
Дерево размера 2 * capacity - 1:
- Листья (capacity-1 ... 2*capacity-2): приоритеты опытов
- Внутренние узлы: суммы приоритетов потомков
- Корень [0]: общий приоритет

Выборка за O(log n):
1. Случайное значение в [0, totalPriority]
2. Спуск по дереву к листу
3. Индекс листа → индекс опыта
```

### 4. Исправление обновления статистики (Episodes counter)
**Файл:** `page.tsx`
- **Проблема:** `setStats(prev => ({...prev, episodes: metrics.episodes}))` не вызывал перерисовку React
- **Решение:** Явно указывать все поля включая `fps: prev.fps` для создания нового объекта
- **Результат:** Episodes теперь обновляется в реальном времени при подборе аптечки

```typescript
// ✅ Правильно:
setStats((prev) => ({
  fps: prev.fps, // ← Сохраняем FPS из предыдущего состояния
  epsilon: metrics.epsilon,
  episodes: metrics.episodes, // ← Теперь обновляется!
  ...
}));

// ❌ Неправильно (не вызывает re-render):
setStats((prev) => ({
  ...prev,
  episodes: metrics.episodes,
}));
```

### 5. Детекция застревания бота
**Файл:** `Bot.ts`
- Добавлен метод `checkStuck()` — отслеживает последние 10 позиций
- Если бот прошёл < 5 единиц за 10 шагов — застрял
- При застревании > 3 раз подряд — 50% шанс случайного действия
- **Результат:** Бот обходит препятствия вместо метания на месте

   - Бот исследует коридоры
   - Собирает аптечки (+5 reward)
   - Аптечки респавнятся через 5 секунд

## Улучшения RL-поведения (Март 2026)

### Проблема
Бот слишком часто уходил за границы экрана и неактивно собирал аптечки.

### Решение

#### 1. Увеличенные штрафы
**Файл:** `config.ts`
- `RL_REWARD_OUT_OF_BOUNDS`: -1 → **-5** (сильный штраф за границы)
- `RL_REWARD_PICKUP_ITEM`: 5 → **10** (двойная награда за аптечки)
- `RL_REWARD_APPROACH_ITEM`: 0.2 → **0.5** (награда за приближение)
- `RL_REWARD_IDLE`: **-0.5** (новый штраф за бездействие)

#### 2. Штраф за бездействие
**Файл:** `Bot.ts`, `update()`
```typescript
const movedDist = Math.sqrt(
  (this.x - prevX) ** 2 + (this.y - prevY) ** 2
);
if (movedDist < 0.5 && this.item) {
  reward += RL_REWARD_IDLE; // -0.5
}
```

#### 3. Улучшенный state
**Файл:** `Bot.ts`, `getState()`
Добавлены 3 значения о направлении к аптечке:
- `dx / dist` — нормализованный вектор X к аптечке
- `dy / dist` — нормализованный вектор Y к аптечке
- `dist / 300` — расстояние до аптечке (0-1)

**Размер state:** 44 → **47**

#### 4. Ожидаемое поведение
- Бот активнее движется (штраф за стояние)
- Бот избегает границ (штраф -5)
- Бот целенаправленно собирает аптечки (+10 за сбор)
- Бот учится двигаться к аптечке (+0.5 за приближение)
