## Настройка параметров

### 1. Скорость вращения
В шейдере строка:
```glsl
uv.x = (angle + uTime * 0.1) / 6.28318 + 0.5;  // 0.1 — скорость вращения
```
**Как менять:**
- `0.05` — медленнее (1 оборот за ~125 секунд)
- `0.1` — текущая скорость (~63 секунды)
- `0.2` — быстрее (~31 секунда)

В Scene3.tsx для анимации:
```tsx
filter.resources.planetUniforms.uniforms.uTime += 0.01;  // шаг за кадр
```

### 2. Сила сферического искажения
В шейдере строка:
```glsl
float distortion = 0.3 * r;  // 0.3 — сила искажения
```
**Как менять:**
- `0.0` — нет искажения (плоский диск)
- `0.2` — лёгкое искажение
- `0.3` — текущее (мягкая сфера)
- `0.5` — сильное искажение

### 3. Положение источника света
В createPlanetFilter:
```ts
lightPos: [0.5, 0.3]  // [x, y] от -1 до 1
```
**Примеры:**
- `[0.5, 0.3]` — свет справа-сверху (по умолчанию)
- `[0.0, 0.0]` — свет по центру (спереди)
- `[-0.5, 0.3]` — свет слева-сверху
- `[1.0, 0.0]` — свет справа по горизонтали
- `[0.0, 1.0]` — свет сверху

### 4. Минимальная освещённость (ночная сторона)
В шейдере строка:
```glsl
light = clamp(light, 0.05, 1.0);  // 0.05 — мин. подсветка
```
**Как менять:**
- `0.0` — полная темнота на ночной стороне
- `0.05` — лёгкая подсветка (по умолчанию)
- `0.1` — более яркая ночная сторона
- `0.2` — очень яркая ночная сторона
